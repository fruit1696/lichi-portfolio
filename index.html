Work

Cloud Asset Management Project at Accenture [Dec 2020 - Aug 2022]
During my tenure at Accenture, I had the privilege to be a part of a transformative project known as Cloud Asset Management (CAM). CAM stands as a pivotal practice for organizations harnessing cloud services, aiming to streamline the management of cloud resources for enhanced cost efficiency, heightened security, and improved operational effectiveness.

Inventory and Discovery: Our journey with the CAM project commenced with a meticulous Inventory and Discovery phase. This involved comprehensive cataloging of all cloud assets and services, ranging from virtual machines to data storage, software licenses, and myriad other cloud resources. This initial undertaking provided crucial insights into the sprawling cloud landscape within the organization.

Cost Management: A cornerstone of the CAM project was dedicated to the management of cloud-related expenditures. Our team delved deeply into tracking and analyzing these expenses with precision, aiming to ensure the judicious utilization of resources. Identifying underutilized or redundant resources paved the way for cost optimization strategies, resulting in substantial savings for the organization.

Security and Compliance: Security and compliance were paramount in our CAM project. Robust security measures, regular vulnerability assessments, and strict adherence to government regulations were essential components. This bolstered the organization's cybersecurity posture while maintaining regulatory compliance.

Resource Optimization: Another significant objective was resource optimization. We focused on fine-tuning resource allocation, employing strategies such as virtual machine rightsizing and resource scaling based on demand. These efforts not only reduced costs but also bolstered system performance and reliability.

Reporting and Analytics: The CAM project entailed the development of advanced Reporting and Analytics tools. These tools offered valuable insights into cloud resource usage, costs, and security, empowering stakeholders to make data-driven decisions, monitor progress, and pinpoint areas for further enhancement.

Governance: Governance was foundational in CAM. Implementing robust policies and procedures ensured the efficient management of access, permissions, and resource provisioning in the cloud environment. This governance framework maintained order and consistency in cloud operations, safeguarding organizational control and compliance.

My experience working on the CAM project at Accenture was immensely fulfilling. It equipped me with profound knowledge of cloud management practices, cost optimization strategies, security protocols, and the significance of data-driven decision-making in the realm of cloud environments. These skills have proven invaluable in delivering tangible benefits to organizations striving to harness the full potential of cloud services while upholding control and efficiency.

Blockchain-Powered
Decentralized Document
Sharing Application

Description:
In this project, I spearheaded the development of a secure, web-based decentralized document-sharing application harnessing the transformative potential of blockchain technology. This innovative solution was conceived to address the critical challenge of document tampering while prioritizing transparency and data security.

Key Achievements:
-Blockchain Integration: We leveraged the Ethereum blockchain's capabilities to ensure the utmost document integrity by storing data hashes instead of the actual content.
-Collaborative Functionality: Our solution empowered users to seamlessly share and collaborate on documents, eliminating concerns about unauthorized alterations.
-InterPlanetary File System (IPFS): To enhance data storage and retrieval efficiency, we incorporated IPFS technology.
-Cutting-Edge Technologies: The tech stack employed included React for a user-friendly interface, NodeJS for robust backend operations, and Ganache for local blockchain testing.

This project not only successfully addressed a crucial issue but also showcased our proficiency in cutting-edge technologies, emphasizing our commitment to delivering secure and transparent solutions.

Counseling Support
System Data Analysis

Description:
This project involved in-depth data analysis using a counseling support system dataset. Employing OLAP (Online Analytical Processing) operations and data cubes, I extracted valuable insights. To ensure data accuracy, ETL (Extract, Transform, Load) techniques were applied for medical record cleansing. Utilizing advanced tools such as WEKA, I performed data clustering and classification to unveil critical patterns and trends, enabling effective decision-making. Furthermore, comprehensive statistical analyses using Tableau facilitated the visualization of complex data relationships.

Key Contributions:
- Data Insights: Leveraged OLAP and data cubes for insightful analysis.
- Data Cleansing: Applied ETL techniques for accurate data.
- Advanced Analysis: Utilized WEKA for clustering and classification.
- Data Visualization: Employed Tableau to visualize complex data relationships.

This project showcases my expertise in data analysis and visualization, underlining the importance of data-driven decision-making in the counseling support domain.
